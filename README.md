> Librería de **utilidades de ficheros y rutas** en **.NET 10**, pensada para consumir desde PowerBuilder.

## 📋 ¿Qué es esto?

Un pequeño conjunto de helpers de **manejo de archivos y rutas** (comprobaciones, nombres,
directorios…) que uso de base en varios ejemplos del blog. **Sin frameworks, sin paquetes externos**:
solo la BCL de .NET. La gracia es tener estas operaciones resueltas y bien probadas para llamarlas
cómodamente desde PowerBuilder como un `dotnetobject`.

## 🛠️ Requisitos

- **.NET SDK 10.0** o superior

## 🚀 Compilar

```bat
dotnet build FileService.csproj -c Release
```

La DLL queda en `bin\Release\net10.0\`.
